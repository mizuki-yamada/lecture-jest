# Jest演習解答例ページ

## 概要

- Jestの演習問題の解答例をReactで作成したものです

## 環境構築

```bash
# 初期設定
npm ci

npm run build

# 起動
npm run dev
```

## 使用技術

- Vite
- React
- TypeScript
- MUI
- Eslint
- Prettier
- Jest
